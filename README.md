# Shoreline-Hazards-and-Habitats

**Overview**

We have seen through many examples in past years about how coastal habitats have protected communities against flooding, but what habitats provide the most protection, what areas need the most protection, and how do we know that these habitats will hold up to protect our communities in the future? This project uses shoreline hazard indexes to tell a story using data that will hopefully provide useful insight for how habitats can be protected or implemented using ecological design. This project will give me an idea of what the future of these habitats looks like and how our communities will be affected. 

**Tools used**

- R/Rstudio
- PowerBi
- Python (Jupyter/Colab)
- Tableau

**Project Structure**
